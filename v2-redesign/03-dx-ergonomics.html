Frictionless session resumption (the /clear problem)

37 /clear commands in 10 days means the user restarts context constantly. V2 treats every session start as a potential resume.

🔄
New session
starts
→
⚡
session-start
hook fires
→
📋
br ready output
injected into context
→
✓
User sees:
"Ready. Next: br-7.3"
Mechanism: Claude Code hooks.sessionStart runs br ready --short and injects the result as a system prompt prefix. If there's an open epic with ready tasks, the first message the user sees is the task title and its acceptance criteria — no hunting, no "where was I?". The don-cape router already has short-circuit logic for bare br IDs — this just surfaces the ID automatically.

Concrete V2 changes

1

Move conform from slash to commit hook

conform ran 7 times, always after committing — too late to be a gate. In V2, conform runs automatically as part of the commit skill's pre-execution step (before git commit is called). User never has to remember it. Violations surface at the moment they matter.

Hook Chain step
2

Add session-start hook that surfaces br state

37 /clear commands = 37 cold restarts with no context. A hooks.sessionStart entry runs br ready --short and prepends the result as a system prompt. The don-cape router sees this output and can immediately load execute-plan. Zero keystrokes to resume.

Hook Resume
3

Gate /pr behind auto-review (not optional)

review ran 12 times, same frequency as /pr — but run order was inconsistent. In V2, the pr skill runs review as step 1 before opening the PR. If review finds blockers, it stops and reports them. The user doesn't need to remember to run review separately; the cost is zero extra keystrokes.

Chain step Auto
4

Drop five zero-use skills; fold into prose answers

analyze-tests, branch, design-an-interface, explain, and find-test-gaps were never triggered. Don't maintain them as separate skills — don-cape can answer explain-type questions inline, branch creation is a one-liner in a hook, and test gap analysis folds into tdd's pre-pass. Fewer skills = fewer routing misses and a smaller surface for failed slash invocations.
